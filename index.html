Revista

Vozes da Rua
O projeto "Revista Vozes da Rua" amplifica as vozes de pessoas em vulnerabilidade, oferecendo um espaço para suas histórias. Utilizamos Adobe Photoshop para editar imagens e Adobe InDesign para diagramação, garantindo um layout harmonioso e de alta qualidade visual.

WEB DESIGN

Site Black's Urbanos
Este projeto recria o site de uma ONG que acolhe e educa crianças vulneráveis, focando em usabilidade e acessibilidade, incluindo recursos para daltônicos. O site possui seções de doação e voluntariado para incentivar o engajamento. Utilizamos Figma para o desenvolvimento.
